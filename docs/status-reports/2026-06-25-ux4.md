**Resume here:** The design-system foundation is in and gate-green. `apps/web/src/ui/` now holds
a **token sheet** (`tokens.css`, dark mode via `prefers-color-scheme`) + a **base layer**
(`base.css`, element-level styling that restyles the whole app and absorbs the old `index.css`
touch-target floor) + a **seed set of primitives** (`Button`, `Badge`, `Card`, `Alert`,
`EmptyState`, `Skeleton`, `Field`/`Input`/`Select`). The **Account Register** is recomposed onto
them in place — no routing change, no flow change, every prior selector preserved — and passes
axe (WCAG 2.2 AA) in **light and dark**. `Dialog` and `Table` were **deferred** (seed-and-grow:
the Register's editor is a shared inline expand, not a modal; converting it would be a flow
change). **Next: `UX3`** — routing + the persistent app shell (React Router · `ADR-0006`).
