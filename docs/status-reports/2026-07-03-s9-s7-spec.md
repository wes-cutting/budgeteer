## 6. Commands & gotchas (cold-start)

```sh
npm run typecheck && npm run lint && npm run format
npm test                 # 393 passing
npm run test:e2e         # 94 Playwright — see the K24-hardened gotcha below
npm run build --workspace @budgeteer/web   # 117.70 KB gz
```

- **e2e must own the stack (K20 + K24):** before `npm run test:e2e`, stop dev servers on
  `:3001`/`:5173` **and verify with `lsof -nP -iTCP:3001 -iTCP:5173 -sTCP:LISTEN`** — killing the
  npm wrapper can orphan the listening node child, and Playwright will silently reuse it against
  the real dev store (dirty empty-app a11y failures **and** fixture pollution). Restart the dev
  servers after.
- Client calendar dates via `apps/web/src/dates.ts` (never `toISOString()`); the server requires
  every user-facing date/month param (EH8). Schema changes are new frozen migrations
  (`0003-…` + registry + `05_DATA_MODEL`). New response types: define in the service, re-export
  from `apps/api/src/contract.ts` (web `import type`s it — lint-enforced).
- S7 build-slice specifics live in [FEAT-S7 §6](../features/pay-periods.md); don't start it before
  the §9 validation verdict.
